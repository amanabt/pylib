==================
PYTHON LIBRARY
==================
This is a collection of python libraries.
-----------------------------------------

**INSTALLATION**


.. code-block:: bash

   pip install py_lib

or you can even install by cloning this repository and 
running.

.. code-block:: python

    sudo python setup.py install

*Link* : http://pypi.python.org/pypi/py_lib
